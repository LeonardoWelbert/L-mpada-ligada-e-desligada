# L-mpada-ligada-e-desligada
Acabo de fnalizei um projeto prático utilizando HTML, CSS e JavaScript (Vanilla JS) . O projeto simula o funcionamento de uma lâmpada, permitindo:  💡 Ligar e desligar 🖱️ Acender ao passar o mouse 🔁 Quebrar ao dar duplo clique 🚫 Impedir funcionamento após estar quebrada .
